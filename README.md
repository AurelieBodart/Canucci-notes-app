# Canucci Notes App

A React + TypeScript + Vite notes application with Tailwind CSS.

## Setup

1. Install dependencies:
```bash
npm install
```

## Running the Application

To start the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:3000` (or another port if 3000 is taken).

## Building for Production

To build the application for production:

```bash
npm run build
```

The built files will be in the `dist` directory.

## Preview Production Build

To preview the production build locally:

```bash
npm run preview
```

## Project Structure

- `src/pages/NotesPage.tsx` - Main notes page component
- `src/components/NoteForm.tsx` - Form component for creating/editing notes
- `src/App.tsx` - Main application component
- `src/index.tsx` - Application entry point

## Technologies Used

- React 18
- TypeScript
- Vite
- Tailwind CSS
